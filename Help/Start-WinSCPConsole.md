# Start-WinSCPConsole

## SYNOPSIS
Starts an iteractive WinSCP Console.

## SYNTAX

```
Start-WinSCPConsole [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Starts an iteractive WinSCP Console.

## EXAMPLES

### EXAMPLE 1
```
PS C:\> Start-WinSCPConsole
```
