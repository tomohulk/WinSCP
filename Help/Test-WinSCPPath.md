# Test-WinSCPPath

## SYNOPSIS
Checks for existence of remote file or directory.

## SYNTAX

```
Test-WinSCPPath -WinSCPSession <Session> [-Path] <String[]> [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
Checks for existence of remote file or directory.

## EXAMPLES

### EXAMPLE 1
```
PS C:\> Test-WinSCPPath -Path /
True
```
